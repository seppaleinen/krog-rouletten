Run all services together
## Docker commands
```
docker-compose up -d (To start servers in background)
docker-compose up --build -d (To build new servers and start in background)
docker-compose down (To shut down and remove servers)
```

To run project in development
```
cd frontend/webapp
python start_tornado (Starta frontend server)
```
